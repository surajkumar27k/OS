Key Features

Hybrid Scheduler

RMS for hard real-time periodic tasks; EDF for non-critical tasks when no critical task is ready.

Energy-Aware DVFS

Two CPU states (HIGH_PERFORMANCE and POWER_SAVER). Laxity-based switching to save energy.

Priority Inversion Demo

Pull example shows simplified priority inheritance behavior for demonstration.
